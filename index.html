Final Year Project · Live R Shiny Application
Portfolio Optimisation App for IDX ESG Leaders
↗ Open Live App

Problem: Investors in ESG-listed companies on the IDX face uncertainty when allocating portfolio weights. Different investors define "acceptable risk" differently, making a one-size-fits-all model inadequate.

Approach: Built an interactive R Shiny web application combining Fuzzy Linear Programming (to handle investor risk preference uncertainty) and Global Minimum Variance (to find the lowest-risk portfolio allocation).

Result: A deployed, publicly accessible application where users can select ESG stocks, adjust risk tolerance, and see optimised portfolio weights and efficient frontier visualisation in real time.

Simulated Project · Branch Performance Analytics
Branch Distribution Performance Dashboard

Problem: In branch operations, teams often need to monitor inventory health, sales movement, delivery performance, and cost efficiency across multiple areas. In practice, that data is often scattered and hard to read at a glance.

Approach: Built a simulated branch distribution dashboard using dummy FMCG/distribution data, demonstrating how operational and commercial metrics can be structured to support branch-level decisions.

Business relevance: This project is structured around the day-to-day responsibilities of an MT Branch Controller: monitoring stock efficiency, spotting commercial patterns, and using data to support operational decisions. All data shown is simulated.
